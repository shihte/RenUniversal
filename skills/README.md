# CTAR Action Judgment Packages (動作判斷包)

此資料夾專門設計用來存放自定義的**動作判斷包**（Action Judgment Packages）。

## 設計概念
未來核心流水線（Pipeline）將會動態掃描與載入此資料夾下的子目錄，每一個子目錄即代表一個獨立的**動作/姿態偵測組件**。

每個動作判斷包內應規劃：
1. **設定檔**：規範該動作需要啟用哪些 MediaPipe 判斷點（例如：雙肩、眼距、軀幹角度等）。
2. **判定邏輯**：定義動作成立的幾何條件與閾值（Thresholds）。
3. **觸發事件**：當條件滿足時應同步更新的狀態或觸發的回呼（Callbacks）。

---

*註：專案底層之基礎架構模組（如相機控制、基礎校準）已全數整合至 `backend/services/` 資料夾中，保持本目錄之純淨，供自定義動作擴充使用。*
