🧠 技能中心引導 (Posture Skills Onboarding)

引導如何啟用、調整與創建自訂動作偵測規則

1. 內建核心技能

RenUniversal 預載了駝背 (Slouch)、軀幹左右偏斜 (Sway) 與前傾 (Lean) 三大核心判定包。您可以隨時調整各項敏感度閾值並即時儲存。

2. 自訂判斷語法

在右側建構器中輸入名稱與描述，然後添加檢測規則。如 torso_sway > sway_threshold，系統將會自動生成 Python 邏輯包。

3. 即時閾值微調

創建後的技能會顯示在左側。拖動閾值拉條時，系統將即時通知後端更新，讓您在不需要重啟程式的情況下微調姿勢判定基準。

自訂動作語法建構器

限小寫英文、數字與底線，不可與現有動作包重名。

🧍 身體 Pose (p前綴)
  • p0: 鼻尖
  • p11: 左肩
  • p12: 右肩
  • p13: 左肘
  • p14: 右肘
  • p15: 左手腕
  • p16: 右手腕
  • p23: 左髖
  • p24: 右髖
👤 臉部 Face Mesh (f前綴)
  • f1: 鼻尖
  • f152: 下巴
  • f33: 左眼外角
  • f263: 右眼外角
  • f133: 左眼內角
  • f362: 右眼內角
  • f13: 上唇中點
  • f14: 下唇中點
  • f4: 鼻梁
💡 f = 臉部 (0-467)，p = 身體骨骼 (0-32)

格式：f1,f152 >< num=20%　說明：點位1,點位2 方向 num=閾值%

可直接編輯語法，支持 AND / OR / NOT 及點位 ID 語法。
