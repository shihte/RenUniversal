🧠 技能中心引導 (Posture Skills Onboarding)

引導如何啟用、調整與創建自訂動作偵測規則

1. 內建核心技能

RenUniversal 預載了駝背 (Slouch)、軀幹左右偏斜 (Sway) 與前傾 (Lean) 三大核心判定包。您可以隨時調整各項敏感度閾值並即時儲存。

2. 自訂判斷語法

在右側建構器中輸入名稱與描述，然後添加檢測規則。如 torso_sway > sway_threshold，系統將會自動生成 Python 邏輯包。

3. 即時閾值微調

創建後的技能會顯示在左側。拖動閾值拉條時，系統將即時通知後端更新，讓您在不需要重啟程式的情況下微調姿勢判定基準。

🎛️ 事件處理中心 (Event Handling Hub)

使用簡單的 boolean 邏輯（AND, OR, NOT）來組合不同的動作偵測狀態，觸發更複雜的複合事件

邏輯語法規則

您可以使用 AND, OR, NOT 與括號來組合動作包。例如：slouch AND sway 代表同時發生低頭與左右偏斜時觸發該事件。

現有可用動作變數 點擊變數可複製名稱

自訂複合事件邏輯

事件邏輯建構器

限小寫英文、數字與底線，不可與現有事件重名。

邏輯運算子說明：
  • AND：左右兩邊的動作均成立才觸發。
  • OR：左右兩邊任一動作成立即觸發。
  • NOT：動作不成立時才觸發。
  • 支援括號 ( ) 來明確優先順序，如 (slouch OR lean) AND sway

可直接編輯語法，支持 AND / OR / NOT 及點位 ID 語法。
